# Hospital-and-Pharmacy-Data-Analysis
This project aims to analyze and present data on patients visits, disease patterns and pharmacy performance.
By leveraging power bi to prepare, structure, analyze and visualize data. The analyses focuses on patient trends,disease patterns,and pharmacy revenue.

The dashboard is designed to help hospital management make quick data driven decisions.

   Project Objectives.
1. Analyze Disease trends overtime.
2. Pharmacy cost breakdown overtime.
3. County and Department comparison.
4. Identify common diseases across counties.

   Tools used
1. Power Query - To clean data and transform data
2. Power Bi - For modeling data,DAX measures and dashboard creation.
3. Gitbash and Github - for version control and Project documentation.

  Dashboard
It contains;

KPI cards containing 

1. Total visits.
2. Total pharmacy revenue.
3. Average length of stay.

Trend analysis.

Diseases Most common in counties.

Pharmacy Analysis.

cost breakdown by drug category.

Insights

Senior citizens are the consumers of most of the medicines compared to the other age groups.

Inpatient and Emergency departments generate the most pharmacy revenue.
